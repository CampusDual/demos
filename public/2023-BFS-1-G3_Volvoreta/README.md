# Requisitos
* Docker (con WSL)
* Git

# Windows
## Iniciar aplicación
```
appmazing-app> run_app.bat
```
## Parar la aplicación
```
appmazing-app> wsl.exe docker compose down
```

# WSL
## Iniciar aplicación
```
appmazing-app$ chmod +x run_app.sh
appmazing-app$ ./run_app.sh
```
## Parar la aplicación
```
appmazing-app$ docker compose down
```
